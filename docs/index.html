About Our Project

Sasdfghjkllkjhgfds. Eventually, anyone who is blind or visually impaired will be able to request a model on a website made primarily with accessibility in mind. Teachers of the visually impaired will also be able to request models to facilitate the learning process. From there, the request will show up in the gallery, where a student from a participating school can take on the request. When the model is finished, the student can ship the model out and fulfill the request.

We hope to extend this project to the greater Cincinnati area, involving a wide variety of schools to participate and to help improve the effort.

Note: for security purposes login to the website is handled through Google Authentication, and school accounts have to be added manually.

Making improvements

Anyone can contribute to the public Github repository, which contains all of the code for the website.
